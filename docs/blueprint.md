# **App Name**: Torah Gems

## Core Features:

- Parsha Selection: Display a list of all Parshiot (Torah portions) for easy navigation.
- Torah Insights Display: Display Torah insights (divrei Torah) for the selected Parsha.
- Admin: New Torah Insights Submission: Allow the admin user to submit new Torah insights for each Parsha.
- Admin: Edit Torah Insights: Allow the admin user to edit existing Torah insights.
- Parshat HaShavua Display: Display the current Parshat HaShavua (Torah portion of the week) on the app's entry screen, showing the latest Torah insight uploaded by the admin.
- Date Display: Always display the Hebrew and Gregorian dates in the corner of the screen.
- Database Storage: Store divrei Torah for each parsha.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking spirituality and tradition.
- Background color: Light gray (#ECEFF1), providing a clean and readable backdrop.
- Accent color: Warm gold (#FFC107) to highlight important information and add a touch of reverence.
- Body font: 'Alegreya', a serif typeface that is both contemporary and intellectual
- Headline font: 'Belleza', a sans-serif that is fashionable and artistic
- Use subtle, meaningful icons to represent each Parsha or category.
- Maintain a clean, organized layout that prioritizes readability.
